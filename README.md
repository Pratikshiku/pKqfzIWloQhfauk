## Online Course Selection System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/1b4425f8e9b84a6389d2943455ea5668/ghb20250916) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript` etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): Regular User (Student), Teacher, Administrator.

**Regular User Functions:**
1. Login
2. Register
3. Change Password
4. Personal Information
5. Course Information Management
6. Course Scheduling Management
7. Course Selection Management
8. Student Grade Management
9. Course Selection Restriction Management
10. Announcement Information Management

**Teacher Functions:**
1. Login
2. Register
3. Change Password
4. Personal Information
5. Student Management
6. Course Information Management
7. Course Scheduling Management
8. Course Selection Management
9. Student Grade Management
10. Course Selection Restriction Management
11. Announcement Information Management



**Administrator Functions:**
1. Login
2. Personal Center (Change Password, Personal Information)
3. Administrator Management
4. Teacher Management
5. Student Management
6. Course Information Management
7. Course Scheduling Management
8. Course Selection Management
9. Student Grade Management
10. Basic Data Management (Class Management, Grade Type Management, Period Management, Course Type Management, Announcement Type Management, Week Management, Semester Management)
11. Course Selection Restriction Management
12. Announcement Information Management
